ID's & Classes | An in-depth look

In web development there are two languages that make up the basics of a website, when you look at a web page the content which includes text and images come from the HTML language, whereas the styling and graphics you see on a web page are from CSS, together these languages work together to create a stunning user experience.

ID's are one of the two ways to target and edit html text with css. In css a ID is selected with a # symbol in fron to of the ID name. ID's are only meant to be used to target a single element.

An example of HTML and CSS working together can be seen below.

HTML


            <div id="id-example">
            
            <h1>Hello World...</h1>
            
            <p>This is a right of passage for developers</p>
            
            </div>

CSS 


            #id-example {
                background-color: #ccc;
                padding: 20px
            }

Classes can be used in HTML to indentify multiple elements, to target a HTML element in CSS with a class selector we use a full stop, an example can be seen below

HTML


            <div class="class-example">
            
            <h1>Classes are used to target multiple elements!</h1>
            
            <p class="lines">This is a single line</p>
            <p class="lines">This is another single line</p>
            <p class="lines">This is yet another single line</p>
            
            </div>

CSS 


            .class-example {
                background-color: #ccc;
                padding: 20px
            }

            .lines {
                font-size:15px;
                font-weight:bold;
            }
