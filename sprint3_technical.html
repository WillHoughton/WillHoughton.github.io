Differences between HTML and CSS

HTML and CSS are a pair that make up every website we see today, they work together to form great websites that are visually appealing.

HTML is the content that we see on a webpage, HTML makes up all of the words on a page, it also includes things such as tables and forms that you may interact with on a website.

CSS is a fun language that can be described as the paint brushes of web development, all of the styling, colours, fonts and many more things that we see on web pages are due to CSS.

Control Flow

Control flow is a terminology in computer programming that is the order in which functions, statements ect are executed.

Loops

Loops are used when you want to run code several times over, the most common way to create a loop is using the for statement, however there are several other loop statements that can be used, they can be seen below.

  • for/in - loops through the properties of an object
  • for/of - loops through the values of an iterable object
  • while - loops through a block of code while a specified condition is true
  • do/while - also loops through a block of code while a specified condition is true

The Dom

The Dom, short for Document Object Model is a useful tool that all the major web browsers have built into them, it allows us, the user, to interact with web pages and make changes on our local version, if you want to interact with the DOM follow these steps below.

How to interact with the DOM

  1. Right Click on the Web page you wish to use the DOM on
  2. Click Inspect
  3. Happy exploring!
